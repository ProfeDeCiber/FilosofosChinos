# FilosofosChinos
Problema de los filosofos Chinos de Dijsktra

La vida de cinco filósofos chinos consta de dos actividades: pensar y comer. A la hora de comer se sientan en una mesa con tantos palillos y boles como filósofos. Para comer, necesitan los dos palillos.

Cuando un filósofo tiene hambre, intenta coger el palillo de la izquierda y si lo consigue, lo intenta con el de la derecha. Si logra los dos palillos toma unos bocados, deja los palillos y sigue pensando.
